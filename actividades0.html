Nomenclatura Orgánica

La nomenclatura orgánica es el sistema de reglas establecido por la IUPAC (Unión Internacional de Química Pura y Aplicada) para nombrar los compuestos orgánicos de manera sistemática y universal. Este sistema permite que químicos de todo el mundo puedan comunicarse sin ambigüedades sobre las estructuras moleculares.

En esta sección, aprenderás las reglas básicas para nombrar hidrocarburos (alcanos, alquenos, alquinos y compuestos aromáticos), practicarás con ejercicios interactivos y podrás reforzar tus conocimientos con juegos educativos.

Guía de Nomenclatura de Hidrocarburos

Alcanos

Reglas de Nomenclatura

  • Cadena principal: seleccionar la cadena continua más larga de átomos de carbono
  • Numerar la cadena para que los sustituyentes tengan los números más bajos posibles
  • Nombrar los sustituyentes alfabéticamente antes del nombre de la cadena principal
  • Los sustituyentes se indican con prefijos (di-, tri-, tetra-) cuando hay más de uno
  • Terminación: -ano para alcanos

Ejemplos

CH₄
Metano
CH₃-CH₂-CH₃
Propano
CH₃-CH(CH₃)-CH₃
2-metilpropano

Alquenos

Reglas de Nomenclatura

  • Seleccionar la cadena más larga que contenga el doble enlace
  • Numerar la cadena para que el doble enlace tenga el número más bajo posible
  • Indicar la posición del doble enlace antes del nombre principal
  • Los sustituyentes se nombran como en los alcanos
  • Terminación: -eno para alquenos
  • Cuando hay más de un doble enlace, usar dieno, trieno, etc.

Ejemplos

CH₂=CH₂
Eteno
CH₃-CH=CH₂
Propeno
CH₃-CH=CH-CH₃
2-buteno

Alquinos

Reglas de Nomenclatura

  • Seleccionar la cadena más larga que contenga el triple enlace
  • Numerar la cadena para que el triple enlace tenga el número más bajo posible
  • Indicar la posición del triple enlace antes del nombre principal
  • Los sustituyentes se nombran como en los alcanos
  • Terminación: -ino para alquinos
  • Cuando hay más de un triple enlace, usar diino, triino, etc.

Ejemplos

HC≡CH
Etino (acetileno)
CH₃-C≡CH
Propino
CH₃-C≡C-CH₃
2-butino

Aromáticos

Reglas de Nomenclatura

  • El benceno es la base del sistema de nomenclatura
  • Los sustituyentes se nombran como prefijos del benceno
  • Cuando hay dos sustituyentes, se usan los prefijos ortho- (1,2), meta- (1,3) y para- (1,4)
  • Para compuestos con sustituyentes complejos, el benceno puede ser considerado como sustituyente (fenilo)
  • Los hidrocarburos aromáticos policíclicos tienen nombres específicos (naftaleno, antraceno, fenantreno)

Ejemplos

C₆H₆
Benceno
C₆H₅-CH₃
Metilbenceno (Tolueno)
C₆H₄(CH₃)₂
Dimetilbenceno (Xileno)
